# Charte Graphique — Rivers Rock (Nordik)

**Concept :** Minimalisme scandinave — vide, espace, typographie.
**Palette :** Blanc `#FAFAFA`, Gris `#E8E8E4`, Ardoise `#4A4A4A`, Noir `#2B2B2B`, Lin `#B8B5A8`
**Typographie :** Inter (tous usages)
**Logo :** Trait horizontal + "rivers rock" minuscules
**Éléments :** Aucune décoration. Ligne d'horizon comme seul motif graphique.
**Mot-clé :** silence, espace, fonctionnel

## Construction du logo

Le logo s'inscrit dans une grille de **80×24** (1U = 4px).


- Le logo s'inscrit dans une grille de 80×24 unités (1U = 4px). 
- Trait horizontal : (0,8) → (80,8), trait 1.5U, couleur lin. 
- Point d'ancrage : ∅2U à l'extrémité droite (x=80, y=8). 
- Texte 'rivers rock' : Inter 5.5U, font-weight 200, tracking 4U, centré à y=21U. 
- Tout en minuscules, tracking large. 
- Zone de protection : 6U de chaque côté. 
- Taille minimale d'impression : 20mm de large.

## Règles d'usage du logo

| Règle | Description |
|-------|-------------|
| **Ne pas déformer** | Le logo ne doit jamais être étiré, compressé ou tordu. Utiliser toujours le fichier SVG source. |
| **Ne pas recolorier** | Les couleurs du logo sont fixes, sauf pour la version monochrome dédiée. |
| **Ne pas pivoter** | Le logo ne doit pas être incliné ou retourné (sauf proposition Grunge où l'irrégularité est intentionnelle). |
| **Pas d'effets ajoutés** | Pas d'ombres portées, dégradés ou contours non prévus dans la charte. |
| **Taille minimale** | Icône seules : ∅30px écran / 10mm print. Logo complet : 60px écran / 20mm print. |
| **Fonds interdits** | Ne pas placer le logo sur des fonds dont le contraste avec ses couleurs est inférieur à 3:1 (WCAG AA). |

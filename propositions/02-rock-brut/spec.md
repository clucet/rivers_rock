# Proposition n°2 — « Rock Brut » (Industriel & Performatif)

**Date :** 24 juin 2026 (refonte identitaire)
**Inspiration :** Énergie rock, patrimoine industriel rouennais, contrastes forts

## Logo

- **Hexagone** contenant "RR" en Anton (pas d'écusson, pas de cercle)
- Texte "RIVERS ROCK" en Anton positionné sous l'hexagone
- Chevron décoratif sous le texte
- Monochrome : blanc sur noir

## Typographie

| Usage | Police |
|-------|--------|
| Titres / Logo | **Anton** (ultra large, agressif) |
| Corps / Badges | **Inter Tight** (condensé, technique) |
| Données / Footer | **JetBrains Mono** (monospace industriel) |

## Éléments graphiques

- **Chevrons** et angles vifs (pas de vagues ni de courbes)
- **Grain argentique lourd** 10% sur tous les fonds
- **Badges pictogram** : carré orange, fond plein, bord blanc épais, numéro blanc
- **Aucun arrondi** — tout est droit (card_r = 0)
- **Aucune ombre portée**
- **Bandes horizontales** fines comme séparateurs
- **Aplats** uniquement (pas de dégradés)
- Traitement noir & blanc contrasté

## Site web

- **Dark mode uniquement**
- Typographie large, agressive
- Sections séparées par des bandes de couleur tranchées
- Hover effects brusques (changement de couleur instantané)
- Photos en noir & blanc, grain visible

## Ambiance

Noir profond, flashs orange, typo massive.
Setlist : style fiche technique roadie. Posters : style affiche de concert punk/rock.
Fond d'écran : textures industrielles (métal rouillé, brique, béton).

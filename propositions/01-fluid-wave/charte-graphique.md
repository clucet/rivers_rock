## Logo
## Construction du logo

Le logo s'inscrit dans une grille de **120×40** (1U = 4px).


- Le logo s'inscrit dans une grille de 120×40 unités (1U = 4px). 
- Vague : courbe de Bézier cubique passant par les points (8,22), (60,20), (112,22). 
- Point d'eau : ∅2.5U positionné au centre de la vague. 
- Texte 'RIVERS ROCK' : Bebas Neue 7U, tracking 3U, centré à y=36U. 
- Zone de protection : 4U de chaque côté. 
- Taille minimale d'impression : 20mm de large.

- **Vague seule** + texte "RIVERS ROCK" en Bebas Neue
- Pas d'écusson, pas de cercle
- Couleur vague : ambre `#D4A843`
- Variante réduite : vague seule (sans texte) pour watermark

## Éléments graphiques
- Vagues de Bézier à enveloppe variable
- Grain overlay 3 %
- Badges ronds (numéro en ambre sur fond blanc)
- Coins arrondis 14 pt, ombre légère
- Dégradés radiaux

## Règles d'usage du logo

| Règle | Description |
|-------|-------------|
| **Ne pas déformer** | Le logo ne doit jamais être étiré, compressé ou tordu. Utiliser toujours le fichier SVG source. |
| **Ne pas recolorier** | Les couleurs du logo sont fixes, sauf pour la version monochrome dédiée. |
| **Ne pas pivoter** | Le logo ne doit pas être incliné ou retourné (sauf proposition Grunge où l'irrégularité est intentionnelle). |
| **Pas d'effets ajoutés** | Pas d'ombres portées, dégradés ou contours non prévus dans la charte. |
| **Taille minimale** | Icône seules : ∅30px écran / 10mm print. Logo complet : 60px écran / 20mm print. |
| **Fonds interdits** | Ne pas placer le logo sur des fonds dont le contraste avec ses couleurs est inférieur à 3:1 (WCAG AA). |

## Logo
## Construction du logo

Le logo s'inscrit dans une grille de **120×120** (1U = 4px).


- Le logo s'inscrit dans une grille de 120×120 unités (1U = 4px). 
- Cercle extérieur timbre : r=54U, stroke-dasharray='4 4'. 
- Cercle intérieur : r=46U, trait blanc 3U. 
- Vague : Bézier quartique traversant le cercle (25,60) → (95,60). 
- RIVERS : arc de cercle supérieur, Bebas Neue 8U, tracking 4U. 
- ROCK : arc de cercle inférieur, Bebas Neue 7U, tracking 5U. 
- Zone de protection : 6U de chaque côté. 
- Taille minimale d'impression : 18mm de diamètre.

- **Écusson** (cercle + vague + RIVERS/ROCK)
- Variante néon : glow `#E85D3A` pour site et vidéos
- Variante timbre disque : anneau concentrique supplémentaire
- Monogramme RR : deux R entrelacés avec vague

## Éléments graphiques
- Vagues sinusoïdales en filigrane
- Grain overlay 5 %
- Halftone dots sur les bords
- Flares lumineux (projecteur de scène)
- Double bordure sur les cartes (blanc + or)
- Badges cercles blancs, numéro en or

## Règles d'usage du logo

| Règle | Description |
|-------|-------------|
| **Ne pas déformer** | Le logo ne doit jamais être étiré, compressé ou tordu. Utiliser toujours le fichier SVG source. |
| **Ne pas recolorier** | Les couleurs du logo sont fixes, sauf pour la version monochrome dédiée. |
| **Ne pas pivoter** | Le logo ne doit pas être incliné ou retourné (sauf proposition Grunge où l'irrégularité est intentionnelle). |
| **Pas d'effets ajoutés** | Pas d'ombres portées, dégradés ou contours non prévus dans la charte. |
| **Taille minimale** | Icône seules : ∅30px écran / 10mm print. Logo complet : 60px écran / 20mm print. |
| **Fonds interdits** | Ne pas placer le logo sur des fonds dont le contraste avec ses couleurs est inférieur à 3:1 (WCAG AA). |

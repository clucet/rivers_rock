## Logo
## Construction du logo

Le logo s'inscrit dans une grille de **200×80** (1U = 4px).


- Le logo s'inscrit dans une grille de 200×80 unités (1U = 4px). 
- Arc principal : parabole (10,70) → Q(100,0) → (190,70), trait 3U, couleur brouillard. 
- Arc secondaire : parabole (10,76) → Q(100,6) → (190,76), trait 1.5U, couleur lumière. 
- Point lumineux : ∅5U positionné au maximum des deux arcs (x=100, y≈33). 
- Zone de protection : 8U de chaque côté. 
- Taille minimale d'impression : 25mm de large.

- **Pont arc** : deux arcs paraboliques superposés
- Cercle lumineux `#FFB703` suspendu entre les arcs
- "RIVERS" en Teko sous l'arc inférieur, "ROCK" décalé à droite
- Variante watermark : silhouette de pont seule

## Éléments graphiques
- Courbes caténaires (paraboliques)
- Symétrie miroir (reflets verticaux)
- Flares lumineux aux intersections
- Grille structurelle subtile
- Grain 4 %

## Règles d'usage du logo

| Règle | Description |
|-------|-------------|
| **Ne pas déformer** | Le logo ne doit jamais être étiré, compressé ou tordu. Utiliser toujours le fichier SVG source. |
| **Ne pas recolorier** | Les couleurs du logo sont fixes, sauf pour la version monochrome dédiée. |
| **Ne pas pivoter** | Le logo ne doit pas être incliné ou retourné (sauf proposition Grunge où l'irrégularité est intentionnelle). |
| **Pas d'effets ajoutés** | Pas d'ombres portées, dégradés ou contours non prévus dans la charte. |
| **Taille minimale** | Icône seules : ∅30px écran / 10mm print. Logo complet : 60px écran / 20mm print. |
| **Fonds interdits** | Ne pas placer le logo sur des fonds dont le contraste avec ses couleurs est inférieur à 3:1 (WCAG AA). |
